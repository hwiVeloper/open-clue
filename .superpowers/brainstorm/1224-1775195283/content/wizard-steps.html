위자드 단계 구성 — 어떤 흐름이 맞나요?

시나리오를 처음 만드는 사람 기준으로, 어떤 순서가 자연스러울까요?

A

메타 → 아이템 → 방 → 포인트 → 검증 → 내보내기

스키마 구조 그대로 순서를 따라가는 흐름. 제목/난이도 설정 먼저, 그 다음 등장 아이템 정의, 그 다음 방을 하나씩 추가하며 각 방의 포인트까지 편집.

화면 흐름
①메타 → ②아이템 → ③방 목록 → ④포인트 편집 → ⑤검증 → ⑥내보내기
B

메타 → 방 구조 → 방별 상세(포인트+아이템) → 검증 → 내보내기

먼저 방의 큰 그림(이름, 설명, 연결)을 잡고, 그 다음 방마다 들어가서 포인트·아이템을 채워 넣는 흐름. 시나리오 설계자 관점에서 자연스러움.

화면 흐름
①메타 → ②방 목록/연결 → ③방 상세(포인트+아이템) × N방 → ④검증 → ⑤내보내기
C

자유 편집 메뉴 방식 (위자드 없음)

메인 메뉴에서 "방 편집", "아이템 편집", "메타 편집" 등을 자유롭게 오가는 방식. 순서 제약 없음. 저장/나가기 언제든 가능.
